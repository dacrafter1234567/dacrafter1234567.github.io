Assignments:

Assignment 1 - Basic HTML

Assignment 1 is a basic refresher on creating webpages using HTML. Some elements introduced here are basic images, text effects, tables, and links.

Assignment 2 - Basic CSS

Assignment 2 builds off of the previous one, introducing CSS to our webpages. CSS allows us to easily decorate and modify our pages to our liking.

Assignment 3 - Page Layout

Assignment 3 introduces flex containers, allowing pages to shift their layouts depending on the user's screen resolution.

Assignment 4 - Recreate CSS Page

Assignment 4 is to recreate a USC webpage, focusing heavily on flex-containers and styling.

Assignment 5 - JavaScript Buttons, Functions, and More

Assignment 5 is an introduction to several basic interactive functions.

Assignment 6 - Conditionals

Assignment 6 focuses on the implimentation of interactive java features, such as text boxes, displaying images, and changing colors

Assignment 7 - Loops

Assignment 7 introduces for loops into websites, allowing features to be created a certain amount of times, or even repeated forever

Assignment 8 - Arrays

Assignment 8 adds arrays, allowing for lists of data to be displayed and interacted with.

Assignment 9 - Columns

Assignment 9 uses arrays, classes, and modals, allowing data to be easily sorted through and displayed.
